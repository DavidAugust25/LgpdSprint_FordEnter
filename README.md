Esse é um projeto feito com muito amor e carinho, respeitando todos os requerimentos da Sprint, com o intuito de tirar uma nota boa. 

Aqui há a lista de todas as mudanças feitas no arquivo disponibilizado pelo professor:

Gerais:

Header
- Margem da logo da Ford ajustada
- Menu de navegação centralizado

Footer
- Reintrodução do texto do copyright
- Reintrodução dos links nos ícones das redes sociais

Específicas:

Main - Index 
- Introdução do carrossel na página
- Introdução dos botões para manipulação do carrossel
- Ajuste da posição de todos os elementos
- Reintrodução dos links na div das imagens
- O link da 'Ford, Nossa História' nos leva ao site oficial da Ford

Main - Lançamentos
- Introdução da tela de comparação, utilizando javascript para administrar as matrizes selecionadas.
- Ajuste na lógica matemática das funções
- Ajustes estéticos na página

Main - Contato
- Remoção dos campos de CPF e Sobrenome, como foi pedido
- Elaborei um método de barrar o preenchimento dos campos utilizando apenas espaços
- Caso os campos estejam preenchidos corretamente, a mensagem de envio será enviada com sucesso.
- Adicionei o console.log para verificar os dados enviados

Estou enviando este código pois ele apresenta qualidade adequada e atende todos os requisitos da sprint.
